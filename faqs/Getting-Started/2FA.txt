Setting Up Two-Factor Authentication (2FA)

Overview
Effective June 30, 2025, all users who have passed an evaluation are required to enable Two-Factor Authentication (2FA) on their Apex account.

Two-Factor Authentication adds an additional security step when logging in and helps protect accounts from unauthorized access.


What is Two-Factor Authentication (2FA)?
Two-Factor Authentication (2FA) is a security feature that requires two forms of verification when logging into your account.

In addition to your password, you must also enter a one-time code generated by an authenticator app on your phone or another device.

Who Must Enable 2FA
All users who have passed an evaluation must enable Two-Factor Authentication in order to continue using their Apex account.

Important details:

- This requirement applies regardless of when the evaluation was passed.
- Once enabled, 2FA remains active on the account indefinitely.
- Disabling 2FA is possible, but doing so will limit certain dashboard features such as Eval Charts and PA Charts.


What Happens After Passing an Evaluation
After passing an evaluation and logging into your dashboard, you will see a notification requiring you to enable Two-Factor Authentication.

You will not be able to take further actions on the account until the 2FA setup process is completed.


How to Set Up 2FA
To enable Two-Factor Authentication:

1. Download an authenticator app if you do not already have one.

Common authenticator apps include:
Google Authenticator
Authy
Microsoft Authenticator

2. Open the authenticator app.

3. Tap the "+" symbol inside the app.

4. Choose the option to scan a QR code.

5. Scan the QR code shown on your Apex Member Dashboard.

6. The authenticator app will generate a 6-digit code.

7. Enter the generated 6-digit code into the setup screen on your Apex dashboard.


Confirmation
Once the correct code is entered, you will see a confirmation message indicating that Two-Factor Authentication has been successfully enabled on your account.


Logging In With 2FA Enabled
After 2FA is enabled, logging into your Apex account requires an additional step.

Login process:

1. Enter your username or email and password as normal.
2. Click the Log In button.
3. When prompted for a One-Time Password, open your authenticator app.
4. Enter the 6-digit code generated by the app.
5. Click Confirm to complete the login.


One-Time Password Information
The One-Time Password is the 6-digit numeric code generated by your authenticator app.

Important details:

- The code changes automatically every time you log in.
- Each code is only valid for approximately 30 seconds.


Troubleshooting 2FA Issues
If you receive an error message related to second-factor authentication, verify the following:

- The correct numeric code from your authenticator app is entered.
- The code has not expired (codes refresh approximately every 30 seconds).
- The correct authenticator entry is used if multiple logins are stored in the app.


Multiple Apex Accounts
If you have both a personal Apex account and a business Apex account:

- You must enable 2FA separately for each account.
- Each account will generate its own unique authenticator code.

When logging in, make sure you are using the authenticator code associated with the correct Apex account.
