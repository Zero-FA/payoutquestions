WealthCharts Setup and Connection

Overview
This guide explains how to set up and use WealthCharts with Apex Trader Funding accounts. It includes instructions for logging in, understanding the Broker Portfolio, using Apex layouts, and setting up the Liquidation Indicator.

Introduction to WealthCharts
WealthCharts provides traders with advanced tools, market research, and trading ideas based on strategies used by experienced traders. The platform is designed to help traders analyze markets, monitor positions, and execute trades efficiently.

Logging Into Your Apex Trader Funding Account

Step 1 — Access Your Apex Account
Log in to your Apex Trader Funding account using the credentials provided after purchasing your evaluation or Performance Account.

Step 2 — Watch the Setup Guide
A video tutorial is provided within the Apex knowledge base that walks through:

• Initial WealthCharts login  
• Platform configuration  
• Connecting Apex accounts  
• Navigating the interface  

The tutorial demonstrates how to properly install and configure WealthCharts to begin trading.

Broker Portfolio

The Broker Portfolio is the primary window used to monitor trading activity. It allows traders to track positions and orders placed through their trading account.

The Broker Portfolio contains two main tabs:

• Positions  
• Orders


Open Positions Tab

The Positions tab displays all currently open positions in the trading account.

Each position contains the following information:

Symbol  
The instrument being traded.

Quantity  
The number of contracts currently held. A negative sign indicates a short position.

Direction  
Indicates whether the position is long or short.

Cost Basis  
The price level at which the position was entered.

P/L (Profit or Loss)  
The current profit or loss of the position.


Orders Tab

The Orders tab displays all orders associated with the account.

This includes:

• Active orders  
• Completed orders  
• Cancelled orders

Active orders will include a Cancel button.

Each order contains the following information:

Symbol  
The instrument being traded.

Date  
The date and time the order was submitted.

Order Type  
The order type used (Market, Limit, or Stop).

Order Details  
The contract size and price level for the order.

Fill Price  
The price at which the order was executed.

Status  
Indicates whether the order is Active, Completed, or Cancelled.

Liquidation Indicator

The Liquidation Indicator is a tool designed to visualize the liquidation point for your trading account.

The tutorial video included in the Apex knowledge base demonstrates:

• How to install the indicator  
• How to configure the indicator  
• How to interpret the liquidation levels

The indicator helps traders understand where their account may be liquidated if losses reach the drawdown threshold.


Benefits of the Liquidation Indicator

The Liquidation Indicator helps traders:

• Identify liquidation risk levels  
• Visualize account drawdown thresholds  
• Manage risk more effectively  
• Make more informed trading decisions

WealthCharts provides a range of tools designed to support Apex traders, including:

By using these tools and tutorials, traders can configure their platform effectively and improve their overall trading workflow.
